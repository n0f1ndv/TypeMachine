## TypeMachine
TypeMachine uses python and curses module and allows you to improve your typing speed and accuracy in your terminal. The moment you run TypeMachine you can start typing the words displayed on the screen. Once you finish you can practice again your last set of words or try another one.

## Usage

## Controls
r - try again current set of words;
n - randomly create new set of words;
q - quit;
